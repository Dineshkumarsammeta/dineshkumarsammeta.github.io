## Reporting a Vulnerability

If you discover a security vulnerability within this project, please report it responsibly.

**How to report:**
- Create a **private security advisory** on GitHub [here](https://github.com/Dineshkumarsammeta/dineshkumarsammeta.github.io/security/advisories).
- Alternatively, you can email the maintainer directly at **dineshkumarsammeta@gmail.com** with details of the issue.

**What to include:**
- A clear description of the vulnerability.
- Steps to reproduce the issue (if applicable).
- Possible impact or affected areas.

**Response timeline:**
- You can expect an initial response within **72 hours**.
- Once confirmed, a fix or mitigation plan will typically be shared within **7–14 days**.
- You’ll be notified once the issue is resolved and a patched version is deployed.

**Note:**  
Please **do not disclose** any vulnerabilities publicly until they have been reviewed and addressed.

Thank you for helping keep this project secure.
